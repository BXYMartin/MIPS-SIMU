# MIPS-SIMU
A Simple (Not Comprehended) MIPS Assembly Simulator for compiler optimizations.

## Instructions
`.globl`, `.data`, `.text`

`add`, `addu`, `addiu`, `sub`, `subu`, `subiu`, `mul`, `div`, `syscall`, `sll`, `j`, `jr`, `jal`, `beq`, `bne` and more...

## Note
`Not All Instructions are implemented`
